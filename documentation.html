Project Goals

The purpose of the project is to demonstrate the skill of building static websites utilizing HTML5, CSS, JavaScript and responsive web design.

My goal for the project was to make a homepage for myself that acts as a portfolio, where I could showcase my school works and projects.

Learning goals I set for myself:

  1. Project planning and management
  2. Content creation for website
  3. Template design with drawing software
  4. Writing html and css code from scratch based on the template
  5. Responsive web design
  6. Adding JavaScript functionality
